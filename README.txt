INSTRUCTIONS POUR DÉPLOYER LE CHATBOT RE-FAP SUR VERCEL

1. Créez un compte gratuit sur https://vercel.com
2. Cliquez sur "Add New Project" puis "Import" (ou glissez ce dossier zip)
3. Laissez Vercel détecter et déployer le projet
4. Une fois en ligne, ajoutez votre clé OpenAI dans les variables d'environnement :
   - OPENAI_API_KEY : [collez ici votre clé API GPT-4]

5. Cliquez sur "Deploy"
6. Votre chatbot est en ligne à l’adresse fournie par Vercel (ex: https://re-fap-chat.vercel.app)
